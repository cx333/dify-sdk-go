# Dify Go SDK 使用说明

基于官方 OpenAPI 规范构建的 Go 语言 Dify SDK，支持 Chat（对话）、Workflow（工作流）、Knowledge（知识库）、File（文件）全部 API。

## 项目结构

```
dify-api/
├── go.work                     # Go Workspace 管理三模块
├── dify-sdk/                   # SDK 核心库（独立 module）
│   ├── config/                 # .env 配置加载
│   ├── client/                 # HTTP 传输层 + 5 组 API Client
│   ├── auth/                   # 多 Key 轮询管理
│   ├── store/                  # 内存元数据存储
│   └── di/                     # dig.Container 依赖注入
├── examples/                   # 示例模块（Fiber 服务 + CLI 演示）
│   ├── main.go                 # Fiber v3 示例服务
│   ├── cmd/demo/               # CLI 演示工具
│   └── .env.example            # 配置模板
├── server/                     # 空白服务端（待业务填充）
│   ├── main.go                 # Fiber v3 空白服务
│   └── .env.example
├── api-specs/                  # Dify OpenAPI 规范（参考）
└── 使用说明.md                  # 本文件
```

## 快速开始

### 1. 配置

复制 .env.example 并填入你的 Dify 实例信息：

```bash
cp examples/.env.example examples/.env
```

编辑 `examples/.env`：

```env
DIFY_BASE_URL=http://localhost:5001/v1    # Dify API 基础地址
DIFY_API_KEYS=app-xxxxxxxx,app-yyyy       # API Key（多个逗号分隔）
DIFY_TIMEOUT=30s                          # 请求超时
DIFY_MAX_RETRIES=3                        # 失败重试次数
SERVER_PORT=3000                          # 服务端口
```

### 2. CLI 快速验证

```bash
# 获取应用信息
cd examples && go run ./cmd/demo/ info

# 发送对话消息
go run ./cmd/demo/ chat abc-123 "你好，介绍一下你自己"

# 查看会话列表
go run ./cmd/demo/ conversations abc-123

# 列出知识库
go run ./cmd/demo/ datasets

# 知识库检索
go run ./cmd/demo/ search <dataset_id> "搜索关键词"

# 执行工作流
go run ./cmd/demo/ workflow abc-123 text="要处理的文本"
```

### 3. 启动示例服务

```bash
cd examples && go run main.go
# 服务启动在 http://localhost:3000

# 测试端点
curl -X POST http://localhost:3000/api/v1/chat \
  -H "Content-Type: application/json" \
  -d '{"query":"hello","user":"test","inputs":{}}'

curl http://localhost:3000/api/v1/info

curl http://localhost:3000/api/v1/datasets

curl -X POST http://localhost:3000/api/v1/workflow \
  -H "Content-Type: application/json" \
  -d '{"inputs":{"text":"hello"},"user":"test"}'
```

## SDK 使用指南

### 配置加载

```go
import "github.com/wgl/dify-sdk/config"

cfg, err := config.Load(".env")
if err != nil {
    log.Fatal(err)
}
```

### 创建 HTTP 客户端

```go
import "github.com/wgl/dify-sdk/client"

httpClient := client.NewHTTPClient(
    cfg.BaseURL,
    cfg.APIKeys[0],
    cfg.Timeout,
    client.DefaultRetryConfig(),
)
```

### Chat API（对话型应用）

```go
chat := client.NewChatClient(httpClient)

// 阻塞模式发送消息
resp, err := chat.SendMessage(ctx, client.ChatRequest{
    Query:  "iPhone 13 Pro 的规格是什么？",
    User:   "user-123",
    Inputs: map[string]any{},
})
fmt.Println(resp.Answer)

// 流式模式
events, errs := chat.SendMessageStream(ctx, client.ChatRequest{
    Query:        "讲个故事",
    User:         "user-123",
    ResponseMode: "streaming",
    Inputs:       map[string]any{},
})
for ev := range events {
    fmt.Print(ev.Answer) // 逐字输出
}

// 会话管理
convs, _ := chat.GetConversations(ctx, "user-123", "", 20)
msgs, _ := chat.GetMessages(ctx, convs.Data[0].ID, "user-123", "", 20)

// 消息反馈
chat.Feedback(ctx, messageID, client.FeedbackRequest{
    Rating: "like",
    User:   "user-123",
})
```

### Workflow API（工作流应用）

```go
wf := client.NewWorkflowClient(httpClient)

// 阻塞模式执行
resp, err := wf.Run(ctx, client.WorkflowRunRequest{
    Inputs: map[string]any{"text": "hello"},
    User:   "user-123",
})
fmt.Println(resp.Data.Outputs)

// 流式模式（含节点事件）
events, errs := wf.RunStream(ctx, client.WorkflowRunRequest{
    Inputs: map[string]any{"text": "hello"},
    User:   "user-123",
})
for ev := range events {
    switch ev.Event {
    case "workflow_started":
        fmt.Println("工作流开始")
    case "node_finished":
        fmt.Println("节点完成")
    case "text_chunk":
        fmt.Print(ev.Answer)
    case "workflow_finished":
        fmt.Println("工作流结束")
    }
}

// 查询执行结果
detail, _ := wf.GetRunDetail(ctx, runID)

// 查询日志
logs, _ := wf.GetLogs(ctx, client.WorkflowLogsOptions{
    Status: "succeeded",
    Page:   1,
    Limit:  20,
})
```

### Knowledge API（知识库）

```go
kb := client.NewKnowledgeClient(httpClient)

// 创建知识库
ds, _ := kb.CreateDataset(ctx, client.CreateDatasetRequest{
    Name:        "我的知识库",
    Description: "产品文档",
})

// 上传文档
resp, _ := kb.CreateDocumentFromText(ctx, ds.ID, client.CreateDocByTextRequest{
    Name: "产品介绍",
    Text: "这是一份产品介绍文档...",
})

// 检索
results, _ := kb.RetrieveSegments(ctx, ds.ID, client.RetrieveRequest{
    Query: "产品功能",
})

// 列出知识库
list, _ := kb.ListDatasets(ctx, 1, 20)
```

### File API（文件操作）

```go
file := client.NewFileClient(httpClient)

// 上传文件
f, _ := os.Open("document.pdf")
uploadResp, _ := file.UploadFile(ctx, "document.pdf", f, "user-123")

// 在消息中使用上传的文件
chat.SendMessage(ctx, client.ChatRequest{
    Query: "总结这个文档",
    User:  "user-123",
    Inputs: map[string]any{},
    Files: []client.InputFile{{
        Type:           "document",
        TransferMethod: "local_file",
        UploadFileID:   uploadResp.ID,
    }},
})

// 下载文件
data, mimeType, _ := file.DownloadFile(ctx, uploadResp.ID, true)
```

### 多 Key 管理

```go
import "github.com/wgl/dify-sdk/auth"

km := auth.NewKeyManager(cfg.APIKeys)

// 轮询获取 Key（自动跳过失效 Key）
for i := 0; i < 10; i++ {
    key := km.Next()
    httpClient.SetAPIKey(key)
    // 使用新 Key 发请求
}

// 标记失效
km.MarkFailed("app-bad-key")

// 获取全部
allKeys := km.All()
```

### 内存存储

```go
import "github.com/wgl/dify-sdk/store"

s := store.NewMemoryStore()

// 预加载所有 Key 的元数据
s.Preload(ctx, cfg.APIKeys, func(ctx context.Context, hc *client.HTTPClient) ([]store.AppMeta, error) {
    chat := client.NewChatClient(hc)
    info, err := chat.GetAppInfo(ctx)
    if err != nil {
        return nil, err
    }
    return []store.AppMeta{{
        ID:   info.Name,
        Name: info.Name,
        Mode: info.Mode,
    }}, nil
}, cfg.BaseURL, cfg.Timeout, 5)

// 查询
app, _ := s.GetByID("app-name")
chatApps := s.GetByMode("chat")
byKey := s.GetByKey(cfg.APIKeys[0])
```

### 依赖注入

```go
import "github.com/wgl/dify-sdk/di"

container, _ := di.BuildContainer(cfg)

// 自动注入
container.Invoke(func(h *client.HTTPClient) {
    chat := client.NewChatClient(h)
    // 使用 chat
})
```

## 错误处理

```go
var difyErr *client.DifyError
if errors.As(err, &difyErr) {
    fmt.Printf("Dify 错误: [%d] %s\n", difyErr.StatusCode, difyErr.Code)
    fmt.Printf("详情: %s\n", difyErr.Message)
}
```

常见错误码：

| Code | 含义 |
|------|------|
| `not_chat_app` | 应用模式不匹配（用 Chat API 调了 Workflow 应用） |
| `not_workflow_app` | 应用模式不匹配（用 Workflow API 调了 Chat 应用） |
| `invalid_param` | 参数缺失或无效 |
| `provider_not_initialize` | 模型提供商未配置 |
| `provider_quota_exceeded` | 模型配额用尽 |
| `too_many_requests` | 并发请求过多（触发限流） |
| `not_found` | 资源不存在 |

## 架构约束

- **配置**：一律使用 `.env` 文件，禁止 config.yaml
- **DI**：所有模块通过 `dig.Container` 组装
- **HTTP 框架**：服务端使用 Fiber v3
- **HTTP 客户端**：标准库 `net/http`，连接池复用
- **认证**：`Authorization: Bearer {API_KEY}`
